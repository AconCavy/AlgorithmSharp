# AtCoderLibraryCSharp

AtCoderLibraryCSharp is the **UNOFFICIAL** AtCoderLibrary for C#.

The library is based on [ac-library](https://github.com/atcoder/ac-library) and written by C#.

- CompressedSparseRow
- Convolution
- DisjointSetUnion
- FenwickTree
- LazySegmentTree
- Math
- MaxFlowGraph
- MinCostFlowGraph
- ModuloInteger

- StringAlgorithm

- TwoSatisfiability

- LowestCommonAncestor
- PriorityQueue
